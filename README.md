#  Decision Tree Classification Project

##  Overview

This project demonstrates the implementation of a **Decision Tree Classifier** using Python and Scikit-learn.
The goal is to build an interpretable machine learning model, understand how decision trees work, and evaluate performance using standard classification metrics.

This project is part of my hands-on learning journey in **Machine Learning and Data Science**, where I focus on building real-world, portfolio-ready projects.
